# Nano Banana — Telegram Mini App

Мини-приложение для генерации изображений в стиле тёмной темы и neumorphism с градиентами в духе Telegram Premium.

## Возможности

- **Главный экран:** поле промпта, кнопка Generate, опции Style / 1:1, HIGH QUALITY, теги стилей
- **Сетка Recent Creations:** 3 колонки (адаптивно), последние арты + слот с индикатором загрузки
- **Прогресс-бар:** отображается во время «генерации»
- **Галерея:** экран со всеми изображениями (View All)
- **Preview:** полноэкранный просмотр с зумом по тапу и кнопками Share / Export
- **Стиль:** тёмная тема, neumorphism, градиенты (фиолетовый → синий), анимации входа и появления элементов

## Запуск локально

1. Разместите файлы на HTTPS-хосте (Telegram Mini App открывается только по HTTPS).
2. Либо используйте локальный сервер и туннель (например, ngrok):

   ```bash
   npx serve .
   ```

3. В [@BotFather](https://t.me/BotFather) создайте бота и задайте Mini App URL: `https://ваш-домен/` или туннель.

## Структура

- `index.html` — разметка, Telegram Web App SDK
- `styles.css` — тёмная тема, neumorphism, градиенты, анимации
- `app.js` — навигация, mock-генерация, прогресс, галерея, preview, share/export

Генерация изображений сейчас **заглушка** (mock): используется прогресс и подставные картинки. Для реальной генерации нужно подключить свой backend/API.
